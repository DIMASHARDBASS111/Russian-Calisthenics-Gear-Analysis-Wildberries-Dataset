# Анализ спорттоваров для калистеники (на основе данных с Wildberries)

**Цель проекта** - проанализировать российский рынок товаров для занятий калистеникой/фитнесом на основе шести выбранных категорий. Выявить ценовые тренды, посмотреть, какие бренды находятся в топе и изучить особенности маркетинга WB.

## Структура репозитория

* `data/` - папка с исходными (`WorkoutProducts.csv`) и очищенными (`Cleaned_Workout_Products.csv`) данными.

* `notebooks/01_data_collection.ipynb` - получение и первичная обработка данных.

* `notebooks/02_data_cleaning.ipynb` - очистка данных и выделение признаков.

* `notebooks/03_visualization.ipynb` - расчет бизнес-метрик, построение графиков и анализ.

* `readme.md` - описание проекта.

## Основные этапы работы

### 1. Сбор данных

Для исследования было выделено шесть категорий товаров: турники, пояса с цепью для отягощения, упоры для отжиманий, фитнес-резинки, брусья, гимнастические кольца. Данные собраны через внутреннее API каталога Wildberries с использованием библиотеки requests. Для каждой категории товаров брались первые 15 страниц с карточками (отсортированными по популярности). В сумме полученный датасет содержит 8300 строк, что немного меньше максимума, так как на каждой странице - максимально количество товаров 100, наибольшее возможное количество строк - 9000 (некоторые категории содержали менее 100 товаров на страницах).

### 2. Очистка данных

В полученном датасете - подавляющее большинство столбцов содержали служебную/техническую информацию. Были оставлены только столбцы, представляющие бизнес-ценность. Для каждого товара были выделены ключевые признаки: обычная цена и цена со скидкой (цены были извлечены из вложенной структуры sizes и переведены из копеек в рубли). Чистые данные были сохранены в новый датасет.

### 3. Анализ и визуализация данных

В финальном ноутбуке выполняется визуальный анализ на основе очищенных данных. Были добавлены новые бизнес-метрики: размер скидки на товар (в процентах) и примерная выручка бренда. Построены boxplot распределения цен по категориям, топ-10 брендов по количеству товаров и продажам, гистограмма скидок, scatterplot зависимости отзывов от цены, а также проанализировано соотношение представленности брендов и их продаж.

## Ключевые выводы

* Фитнес-резинки - наиболее дешёвая и массовая категория, турники занимают средний ценовой сегмент, а брусья остаются самой дорогой.

* Скидки 60-80% стали маркетинговым стандартом и часто создают иллюзию выгоды.

* Бренд Absolute Champion лидирует по продажам, а широта ассортимента не гарантирует высокого объёма продаж.

* Основной спрос сосредоточен в низком ценовом сегменте до 2000 руб.
